実行手順

1. main.cpp（dictionary/dictionary/main.cpp）をC++コンパイラでビルドしてください。
2. dictionary dictionary/dictionary/data/dictionary-data.txt と、コマンドラインで実行してください。
3. 標準出力に"上手"と出力されれば、正常に動作しています。
注意点
・ソースコードのエンコードはUTF-8です
・dictionary/dictionary.xcodeprojは、Mac用のプロジェクトファイル（Xcode 9.3.1）のため、他の環境では使用できません

データファイル仕様
・ファイル形式は、UTF8エンコードのテキストファイル
